ANALYSIS

Here you can find some of the work we develop on performance analysis of algorithmic architectural models

Structural

ADA's algorithmic design tools allow for a smooth comunication with structure analyis tools. Just as one single program can be interpreted by CAD and BIM applications to generate similar geometries in each one, the same is possible with analysis tools such as Robot, for instance.

Images: Shenzhen Bao’an International Airport
Analysis: Maximum Vertical Displacement
Original Project by Studio Fuksas







Lighting

Lighting conditions, such as illuminance, luminance, solar radiation, glare, can have a significant impact on visual comfort and work productivity. Lighting analysis tools give back information retrieved from the simulations they perform. Using an algorithmic-based approach, the results are simultaneously transferred to the program, where they can be used in the modeling process, or further evaluated in an optimization process.

Project: Façade for an Office Building | Shanghai 2017
Analysis: Useful Daylight Illuminance (UDI)
