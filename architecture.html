Case Study projects by ADA

Here you can find some of the projects made using our tools and design approach

ARCHITECTURE

Using algorithms as the medium to explore and develop architectural design ideas, it is possible to achieve Generative Architecture. Using a pure programming-based representation of the intended design not only speeds up the handling of changes that always occur as the design evolves, but also allows for a much larger solution space derived from the formalized representation.

Parametric Modelling

In the past years, there has been an increased use of programming tools in architectural design, not only because it allows architects to achieve nouvelle shapes, but also allows the architects to rapidly experiment and vary their designs. ADA has developed textual parametric design tools which allowed scaling of architectural designs in a more rapid and efficient way when compared with visual tools.

COMPLEX GEOMETRIES

Using algorithmic processes as agents for form generation, a wide range of complex geometries can be explored and manipulated, including shapes which would have been difficult to explore using traditional manual means.

Facade Design

Nowadays, architectural facades are characterized by complex shapes and patterns, mainly due to the use of new design tools, such as Generative Design, which promote a greater design exploration. Using these tools, we developed a computational framework for the generation and exploration of facade designs. The framework includes a classification of facades that helps the designer identify the set of algorithms that best suits the needs of his design intent. By combining the various algorithms provided by the framework, the designer can more easily explore the solution space of the intended design. The following examples were all generated using the framework.

A-BIM: Algorithmic-Based Building Information Modelling

A-BIM explores the Generative Design approach with Building Information Modelling (BIM) in the context of architectural design. This approach offers great advantages to architectural design allowing models to more flexible while containing building information relevant for construction.
