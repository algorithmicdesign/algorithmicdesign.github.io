Generative Design Tools

Here you may find the Generative Design tools for Architecture created by our group
You can install them and learn how to use use them for your own projects and Generative Design experiments

ROSETTA

Rosetta is a generative design tool that allows the creation and manipulation of geometric data in CAD and BIM applications using different programming languages. Its aim is to aid architects in use computational design processes to generate and manipulate complex geometry. At the moment, Rosetta is implemented in AutoCAD, Rhino, SketchUp, Revit, ArchiCAD, OpenGl and TikZ, using Racket, Python, Processing, and Lisp as front-end languages.

  • ROSETTA WEBSITE

KHEPRI

Khepri is an improved generative design tool based on Rosetta that revises the way users interact with the design tool. Khepri possesses the same features as Rosetta, allowing architects to create and manipulate geometric data in both CAD and BIM applications, using different programming languages. Khepri is still in early development but we are already developing the support for all the back-ends and programming languages supported by Rosetta, as well as other languages that prove to be appropriate for beginners and designers.


LUNA MOTH

Luna Moth is a HTML5/JavaScript-based web application that harnesses the performance and graphical capabilities available in modern web browsers, complemented with a companion application for allowing models to be exported to traditional CAD applications. Using this IDE, architects can write their GD programs and visualize the results without being chained to a particular computer, and can easily integrate results into their normal workflow.
